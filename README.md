# Luxora Cleaning Services

A premium, mobile-responsive Next.js and Tailwind CSS website for a luxury cleaning service brand.

## Run Locally

```bash
npm install
cmd /c npm run dev
```

Open `http://localhost:3000`.

## Sections

- Sticky header and luxury hero
- Services
- Why choose us
- Before/after gallery
- Booking process
- Premium pricing cards
- Testimonials
- FAQ
- Quote/contact form
- Footer and sticky WhatsApp CTA
